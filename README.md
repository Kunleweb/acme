This project involvees the development of centralized, well-structured data platform. It involves the following:

- Data Integration : Consolodiation of data into a warehouse
- Data Transformation : Implement staging and gold layers
- Data Modelling : create fact and dimension tables
- Aggregation Layer : Build summarized datasets
- Data Quality & Testing : Ensure accuracy through validations
- Analytics Enablement : Make data usable for reporting


Tech Stack:
- Data Warehouse: Snowflake
- Transformation and Modelling tool: DBT
- Data Ingestion: Airbyte
- Storage: Googledrive
